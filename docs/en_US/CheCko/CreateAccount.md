# Create account

After installation, CheCko will be run in chrome extension environment. But if you don't have any account created or you just install it, it'll open a new tab in chrome as a tutorial of account creation.

### Create password

Your password must contain letters, digits, and symbols. You must store your password properly. CheCko is not able to recover your password if your forget it due to the it only store cipher text in the wallet.

### Backup and validate mnemonic

Currently the mnemonic can only recover the first generated account. If you create new account in the wallet again, you cannot recover it with this mnemonic currently. You must store your mnemonic properly outside CheCko wallet (may be write it down with a piece of paper). CheCko is not able to recover your account if you lose your mnemonic.

### Backup private key

Besides the mnemonic, you also can backup private keys for all of your accounts in CheCko. Be careful don't leak your private keys to anyone else! People who have this private key can access any assets in this account.
